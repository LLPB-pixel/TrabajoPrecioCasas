# TrabajoPrecioCasas

Trabajo en grupo para el curso de IA de Samsung Innovation Campus

## Descripción del proyecto
El objetivo es crear un programa que lea una lista de precios de casas y las muestre ordenadas por precio. El programa debe permitir al usuario ingresar los datos de varias casas, incluyendo su nombre, dirección, número de habitaciones, superficie, precio, etc., y luego mostrarlas en orden ascendente según el precio.

## Herramientas utilizadas
- Python
- Pandas para manipular la información
- Matplotlib para visualizar los resultados
